# Asanoso

A single-page site with a locked "You don't have access to this page" screen and a
placebo access-request form (it looks like it sends a request, but nothing actually
leaves the page — no network calls, no storage).

The main feature is the **Discord / Open Graph link embed**: when the URL is pasted in
Discord it unfurls as:

> **Asanoso BETA, the Free AI**
> Use Asanoso today for completely free. (yez its a small model its not very powerful don't mind it) - shrekis.life hosting

## Files

- `index.html` — the entire site (markup, styles, and the placebo script are all inline).

## Hosting

Served as a static page via GitHub Pages.
